# Universal CommandSpec Fixtures and MCP Mapping Tests

## Parent Initiative
KAT-I-0006 — Agent Platform Strategy Pattern.

## Goal
Provide a single source-of-truth file declaring the four universal `CommandSpec` instances, plus a contract test that asserts every adapter's `install()` exposes them with identical ids and `handlerHint.mcpTool` values.

## Deliverables
- `src/platform/_shared/universal-commands.ts` exporting `UNIVERSAL_COMMANDS: CommandSpec[]`.
- `tests/platform/universal-commands.contract.test.ts` — parameterized over all three adapters from the registry.

## Architecture & Contracts
```ts
export const UNIVERSAL_COMMANDS: CommandSpec[] = [
  { id: "katana-decompose", description: "Decompose a parent doc into children",
    argsSchema: { type: "object", required:["parent"], properties: { parent: { type:"string" } } },
    handlerHint: { mcpTool: "decompose_document" } },
  { id: "katana-work", description: "Start working a task",
    argsSchema: { type: "object", required:["task"], properties: { task: { type:"string" } } },
    handlerHint: { mcpTool: "list_documents" } },
  { id: "katana-board", description: "Show the kanban board",
    argsSchema: { type: "object", properties: { level:{type:"string"}, phase:{type:"string"} } },
    handlerHint: { mcpTool: "list_documents" } },
  { id: "katana-validate", description: "Validate a document",
    argsSchema: { type: "object", required:["short_code"], properties: { short_code: { type:"string" } } },
    handlerHint: { mcpTool: "validate_document" } },
];
```
Each adapter's `install()` MUST call `registerCommand(spec)` for every entry in `UNIVERSAL_COMMANDS` (or equivalent code path) such that the returned `InstallReport.commands` deep-equals `UNIVERSAL_COMMANDS.map(c => c.id)`.

## Do / Don't
- DO drive adapters from this constant — no hardcoded copies of command ids inside adapters.
- DON'T add platform-specific commands here.

## Acceptance
- Contract test passes for `claude-code`, `cursor`, `openai-codex`.
- `mcpTool` values match tools declared in KAT-A-0001 (`decompose_document`, `list_documents`, `validate_document`).

- Blocked by: KAT-T-0140, KAT-T-0141.

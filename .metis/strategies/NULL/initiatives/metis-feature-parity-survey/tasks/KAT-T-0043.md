# Survey metis agent-integration layer (templates, presets, agent rules, code-index)

## Parent Initiative

[[KAT-I-0008]]

## Goal

Survey the four agent-integration capabilities of metis — document templates / required-section enforcement, configuration presets (`full`, `streamlined`, `direct`), agent-rules generation (`CLAUDE.md` injection and `metis cursor` Cursor rules), and the `code-index` skill / `.metis/code-index.md` artifact — and produce one decision row per capability. This is the most Katana-relevant task: templates and agent rules are central to the vision.

## Deliverables

- `/Users/danielcassil/Code/katana/.metis/strategies/NULL/initiatives/metis-feature-parity-survey/notes/survey-agent-integration.md`

## Survey criteria

For **each** capability below, answer all of these questions in order:

1. **What does metis provide?** Behavior description grounded in observed artifacts (template files, generated rule files, config.toml).
2. **How is it implemented?** Surface (CLI subcommand, skill name, on-disk file path, frontmatter convention). Cite exact names.
3. **Evidence sources.** Paths and command outputs. For templates, cite at least one concrete example task or initiative file in the boilerplate workspace.
4. **Katana need.** MVP / Post-MVP / Never. Note: Katana's two-pass model and gates make templates a hard MVP requirement; presets/code-index are weaker fits.
5. **Decision.** Inherit | Depend | Defer | Skip.
6. **Rationale.** 2–4 sentences. For agent rules, explicitly note that Katana wants a *pluggable* platform adapter (Claude / Cursor / Codex), so an inherit may need restructuring.
7. **If Inherit:** which metis source / template files / skills to copy.
8. **Risks / open questions.** Bullets.

### Capabilities in scope

- **C1: Document templates and required-section enforcement** — the markdown skeletons metis injects on `metis create`, required headings, frontmatter shape.
- **C2: Configuration presets** — `metis config` with `full` / `streamlined` / `direct` presets; `--strategies` / `--initiatives` toggles; `config.toml`.
- **C3: Agent-rules generation** — `CLAUDE.md` injection on `metis init`, `metis cursor init` / `refresh` / `update` / `status` for `.cursor/mcp.json` and `.cursor/rules/metis-*.mdc`.
- **C4: Code-index** — the `metis:code-index` skill and the `.metis/code-index.md` artifact format.
